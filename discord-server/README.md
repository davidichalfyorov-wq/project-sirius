# Project Sirius — Discord Server Kit

Полный комплект для Discord-сервера Project Sirius: скрипт-конструктор, который сам строит сервер (роли, каналы, права, иконка, AutoMod), и кастомный бот **Sirius Bot** с антиспамом, рейд-защитой, модерацией, верификацией и системой предложений.

Сервер строится на английском языке в тематике Freelancer: участники — пилоты, модераторы — Station Police, общение — в «баре».

## Что получится

**Роли** (сверху вниз): 👑 Admiral (админ) · 🛡️ Station Police (модераторы) · 🤖 Bots · ⚙️ Shipwright (разработчики) · 🎨 Artist · 🧪 Test Pilot · 💫 Ace (актив) · 🚀 Pilot (верифицированный) · 🌑 Docked (новичок) · 📡 Comm Relay (пинг анонсов).

**Каналы**: 📡 STARPORT (welcome, rules + кнопка верификации, announcements, dev-blog) · 🍻 THE BAR (general, off-topic, media, memes) · 🚀 PROJECT SIRIUS (sirius-chat, engine-dev, content-modding, bug-reports-форум, testing, suggestions, showcase) · 🔊 COMM CHANNELS (голосовые + AFK) · 🛡️ STAFF (staff-chat, mod-log, bot-log).

**Защита**: 4 нативных правила Discord AutoMod (спам, массовые упоминания, мат/оскорбления, скам-ссылки) + собственный антиспам бота: флуд, дубликаты, капс, эмодзи-флуд, инвайт-ссылки, эскалация наказаний (предупреждение → таймаут 10 мин → таймаут 6 ч → кик) и автоматический **рейд-режим** при наплыве новых аккаунтов.

## Установка (один раз, ~10 минут)

Нужен Python 3.10+ (`python --version`).

**1. Создайте приложение бота.** Откройте https://discord.com/developers/applications → *New Application* → имя `Sirius Bot`. Загрузите аватар (наша иконка из `../assets/sirius_icon.png`).

**2. Настройте бота.** Вкладка *Bot*: нажмите *Reset Token* и сохраните токен (показывается один раз!). Ниже включите **оба** переключателя: *Server Members Intent* и *Message Content Intent*.

**3. Пригласите бота.** Вкладка *OAuth2 → URL Generator*: отметьте scopes `bot` + `applications.commands`, в правах — `Administrator`. Откройте сгенерированную ссылку и добавьте бота на ваш сервер (создайте пустой сервер, если его ещё нет).

**4. Узнайте ID сервера.** В Discord: Настройки → Расширенные → включить «Режим разработчика», затем ПКМ по серверу → «Копировать ID сервера».

**5. Заполните `.env`.** В этой папке: скопируйте `.env.example` в `.env`, вставьте токен и ID.

**6. Установите зависимость и постройте сервер:**

```bash
pip install -r requirements.txt
python setup_server.py
```

Скрипт создаст всё и выведет постоянную ссылку-приглашение. Запуск повторно безопасен — он обновляет существующее, а `--wipe` сначала удаляет все каналы/роли (спросит подтверждение).

**7. Запустите бота:**

```bash
python -m bot.main
```

Бот должен работать постоянно — пока он запущен, действуют антиспам, кнопка верификации и команды. Варианты 24/7: свой ПК/сервер, VPS, Raspberry Pi. На Linux удобен systemd или `tmux`.

## Команды бота

`/warn add|list|clear` · `/timeout` `/untimeout` · `/kick` `/ban` `/unban` · `/purge` · `/slowmode` · `/lockdown` · `/raidmode` — модерация (видны только модераторам).
`/suggest` — идея с голосованием 👍/👎 и тредом; `/verdict` — вердикт стаффа; `/poll` — опрос до 5 вариантов.
`/ping` `/serverinfo` `/userinfo` `/avatar` `/say`.

## Настройка

Всё в `config.json`: названия ролей/каналов, цвета, права, тексты правил и приветствия, пороги антиспама (`antispam`), правила AutoMod (`automod`). После изменения структуры запустите `python setup_server.py` ещё раз; изменения настроек антиспама подхватываются перезапуском бота.

Иконка сервера берётся из `assets/sirius_icon.png` рядом со скриптом либо из `../assets/sirius_icon.png` (корень проекта — там лежит оригинал).

## Важно

Никому не передавайте токен из `.env` — это полный доступ к боту. Если токен засветился, сразу сделайте *Reset Token* в Developer Portal. Данные бота (предупреждения, счётчик предложений) хранятся в `bot/data/*.json`.
